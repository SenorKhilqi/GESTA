Apa itu Aplikasi Belajar SIBI?

Aplikasi Belajar SIBI adalah platform web interaktif yang dirancang untuk membantu individu mempelajari dan berlatih Sistem Isyarat Bahasa Indonesia (SIBI). Aplikasi ini menggunakan teknologi computer vision dan machine learning untuk mengenali isyarat tangan secara real-time dan menerjemahkannya menjadi teks, menampilkan visualisasi isyarat dari teks, serta menyediakan permainan edukatif untuk meningkatkan kemampuan mengeja menggunakan SIBI.

Bagaimana Cara Kerjanya?

Sistem kami memanfaatkan teknologi berikut yang berjalan langsung di browser Anda:

  • Computer Vision (MediaPipe Hands): Kami menggunakan MediaPipe Hands dari Google untuk mendeteksi dan melacak titik-titik penting (landmark) pada tangan Anda secara real-time melalui webcam (untuk fitur Isyarat ke Teks).
  • Machine Learning (ONNX.js): Sebuah model machine learning (Random Forest Classifier) yang telah dilatih sebelumnya digunakan untuk mengenali pola isyarat tangan berdasarkan landmark yang terdeteksi. Model ini berjalan di browser menggunakan ONNX.js.
  • Visualisasi Isyarat: Untuk fitur Teks ke Isyarat, aplikasi menampilkan gambar atau animasi yang merepresentasikan isyarat SIBI.
  • Teknologi Web Standar: Antarmuka pengguna (frontend) dibangun menggunakan HTML, CSS, dan JavaScript murni.

Fitur Utama

Isyarat ke Teks

Gunakan webcam Anda untuk menerjemahkan isyarat abjad SIBI menjadi teks secara real-time. Fitur ini sempurna untuk melatih kemampuan Anda dalam melakukan isyarat.

Teks ke Isyarat

Masukkan teks dan lihat representasi visual dari isyarat SIBI yang sesuai. Sebuah cara yang bagus untuk mempelajari bagaimana melakukan isyarat untuk kata ke isyarat abjad SIBI.

Permainan Eja Kata

Sebuah permainan edukatif yang menyenangkan untuk menguji dan meningkatkan kemampuan Anda dalam mengeja kata-kata menggunakan isyarat abjad SIBI.

Mengenal SIBI

Sistem Isyarat Bahasa Indonesia (SIBI) adalah salah satu sistem isyarat yang digunakan di Indonesia untuk mendukung komunikasi bagi teman Tuli dan individu dengan hambatan pendengaran. SIBI memiliki struktur dan kosakata tersendiri yang penting untuk dipelajari.

Mempelajari SIBI membuka pintu komunikasi yang lebih luas dan inklusif.

Detail Teknis

Proyek ini berjalan sepenuhnya di sisi klien (browser pengguna) dan menggunakan:

  • HTML5, CSS3, JavaScript: Untuk membangun antarmuka dan logika aplikasi web.
  • MediaPipe Hands (JavaScript): Untuk deteksi landmark tangan.
  • ONNX.js: Untuk menjalankan model machine learning (Random Forest yang dikonversi ke ONNX) di browser.
  • Dataset: Model dilatih menggunakan dataset gambar isyarat tangan SIBI bersumber dari platform Kaggle.
